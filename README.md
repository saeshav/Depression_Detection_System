# Depression_Detection_System
Detect depression in users Twitter Post and display prediction.
## Requirements

- ftfy
- matplotlib
- nltk
- numpy
- pandas
- keras
- gensim
- flask
- flask_mysqldb
- mysqlclient
- tensorflow==2.11.0
- keras-preprocessing
